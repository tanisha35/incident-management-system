# User Stories

## Customer

- As a customer, I want to register an account so that I can access the application.
- As a customer, I want to log in securely using my credentials.
- As a customer, I want to create a support ticket when I encounter an issue.
- As a customer, I want to view all the tickets I have created.
- As a customer, I want to view the status of my tickets.
- As a customer, I want to update or delete my ticket before it is assigned.
- As a customer, I want to add comments to my ticket to provide additional information.

## Support Engineer

- As a support engineer, I want to view tickets assigned to me.
- As a support engineer, I want to update the status of assigned tickets.
- As a support engineer, I want to update ticket priority when required.
- As a support engineer, I want to add comments while resolving incidents.

## Administrator

- As an administrator, I want to manage user accounts.
- As an administrator, I want to assign tickets to support engineers.
- As an administrator, I want to monitor all tickets in the system.
- As an administrator, I want to update ticket details whenever necessary.
- As an administrator, I want to generate dashboard statistics.
